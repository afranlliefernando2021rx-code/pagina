Tienda de camisetas Retro
Descripción: 
Este proyecto consiste en una tienda web de camisetas de fútbol de estilo retro desarrollado como trabajo final. 
La página permite navegar entre diferentes secciones, visualizar fotos de camisetas y jugadores icónicos, acceder a los detalles de cada una y simular una compra mediante un formulario. 
Tecnologías utlizadas: 
HTML 5, CSS, JavaScript, Bootstrap 5
Cómo ejecutar el proyecto:
1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto. 
3. Ejecutar el archivo 'index.html' desde un navegador.
Funcionalidades:
Navegación entre secciones mediante la barra superior, Catálogo de selecciones, Catálogo de equipos, Detalle de cada camiseta, Carrito de compras, Formulario de compra, Resumen de compra.
